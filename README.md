# HR-analytics-dashboard
# Overview
This project focused on analyzing employee data to uncover key insights related to workforce trends, attrition, and overall HR performance. The dashboard provided a clear and interactive way to understand employee behavior and support data-driven decision-making.
# Objectives
Analyzed employee data to identify patterns in attrition
Evaluated key HR metrics such as employee count, attrition rate, and demographics
Built an interactive dashboard to visualize workforce insights
Enabled better decision-making for HR strategies
# Tools & Technologies Used
Power BI – for dashboard development and visualization
Power Query – for data cleaning and transformation
Excel / Dataset – as the data source
# Key Features
Attrition analysis based on department, age group, and job role
Employee distribution by gender, education, and salary slab
Interactive filters for dynamic data exploration
KPIs such as total employees, attrition count, and attrition rate
# Conclusion
The HR Analytics Dashboard successfully transformed raw employee data into meaningful insights. It helped in identifying critical factors affecting attrition and supported strategic HR decisions.
